# AGENTS.md

## 用户交互偏好

- **执行模式**：自动执行。不要每次都询问确认，直接执行操作。
- **询问原则**：只有在涉及以下情况时才询问用户：
  - 删除重要文件或数据
  - 修改生产环境配置
  - 执行可能有安全风险的操作（如暴露密钥、修改权限等）
  - 用户明确要求你做选择时
- **默认行为**：对于常规的代码修改、文件创建、git操作、依赖安装等，直接执行，无需确认。
